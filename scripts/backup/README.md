# Backup Scripts

See [docs/BACKUP_SYSTEM.md](../../docs/BACKUP_SYSTEM.md) for complete backup documentation.

Scripts in this directory:
- **backup-manager.sh** - Full and daily backups
- **restore-backup.sh** - Restore from backup files
- **verify-backup.sh** - Verify backup integrity
- **monthly-restore-test.sh** - Monthly DR testing
- **rsync-backup-remote.sh** - Remote backup sync
- **rsync-restore-remote.sh** - Remote restore
